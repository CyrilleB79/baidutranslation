# 百度翻译

一个提供百度翻译的NVDA插件。

## 键盘快捷键：

* 翻译刚才听到的内容：NVDA+W
* 反向翻译刚才听到的内容：NVDA+SHIFT+W
* 循环切换自动翻译模式（禁用、常规、反向）：NVDA+F8
* 正向翻译剪贴板中的内容：NVDA+CONTROL+W
* 反向翻译剪贴板中的内容：NVDA+CONTROL+SHIFT+W
